### Retrieve a fiscal year

```http
GET /invoices/cnje_memberships/fiscal_years/:fiscal_year HTTP/1.1
```

```json
{
  "fiscal_year": {
    "year": "2015",
    "deposit_archive_url": "",
    "balance_archive_url": "",
    "total_deposit_amount_in_cents": 0,
    "total_balance_amount_in_cents": 0
  }
}
```
